# Product

## Register

brand

## Users

Codex and agent-skill users who want repeatable image workflows; designers and community operators who need branded sticker packs; AI and Web3 teams that publish messaging assets across chat platforms.

They arrive on the repository page with little context. They need to understand the result, trust the workflow, install the Skill, and run a useful prompt within a few minutes.

## Product Purpose

Sticker Pack Maker turns one character reference into a coherent set of transparent PNG stickers. It handles scene planning, character consistency, exact bubble copy, interrupted generation, chroma cleanup, validation, naming, and ZIP packaging.

Success means a visitor can see the finished quality before reading the implementation, understand why the workflow exists, and reproduce it without reconstructing the process from scattered notes.

## Brand Personality

Precise, capable, playful.

The character art supplies warmth. The surrounding presentation behaves like a production tool: specific claims, visible output, short instructions, and verifiable quality checks.

## Anti-references

- Generic AI project pages that lead with a long feature list.
- Developer-tool pages that use terminal styling as a substitute for identity.
- Inflated claims about one-click automation without showing constraints or validation.
- Meme-heavy Web3 presentation that weakens trust outside that audience.
- Editorial or SaaS templates with interchangeable gradients, card grids, and vague copy.

## Design Principles

1. Show the pack before explaining the pipeline.
2. Make the first successful command easy to find and copy.
3. Separate creative generation from deterministic production checks.
4. Use evidence such as transparent outputs, tests, CI, and file contracts.
5. Keep the character playful and the system around it disciplined.

## Accessibility & Inclusion

Use semantic Markdown, descriptive alt text, readable badge labels, and meaningful link text. Do not encode status only through color. Keep diagrams understandable as text and avoid decorative animation that cannot be disabled.
